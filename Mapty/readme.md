此程式為練習實作用 Javascript 來操作Object 以及實現物件導向程式設計

1.練習物件導向設計(實作一父類別可以操作：
	1.抓取地圖套件
	2.取得點擊位置坐標
	3.show/hide 輸入Form
	4.新增與顯示鍛練的位置點與List
   )
   子類別：
   產生計算跑步與騎車的鍛練位置與資訊
2.練習地圖第三方套件實作
3.練習localstroage的使用
